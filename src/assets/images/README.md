# Product Images

Place your product images here and update `src/assets/products.json` paths accordingly, e.g.:

- `src/assets/images/lavender.jpg`
- `src/assets/images/vanilla.jpg`
- `src/assets/images/citrus.jpg`
- `src/assets/images/seasonal.jpg`

For best results, use optimized JPG/PNG around 1200x900.
